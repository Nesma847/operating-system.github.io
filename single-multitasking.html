    Multi Tasking

    It's an operating system allows the running of more than one program in competition. This is achieved through time-sharing, in which the processor time available is divided between multiple processes. Those processes are repeatedly interrupted in time slices by the operating system's task-scheduling subsystem. It can be defined in the forms of preemptive and cooperative.The operating system slices the CPU time in preventive multitasking, and dedicates a slot to each of the programs. Unix-like operating systems such as Solaris and Linux – as well as non-Unix-like systems such as AmigaOS – support multitasking prevention. Cooperative multitasking is achieved by relying on each process to give a defined amount of time to the other processes

    Single User

    There is one operator / programmer in one user This person may have one or more stations (consoles or onPC). If there is only one user, then all stations will be logged in as the same user. The stations are all synchronized and share the different views and all stations have the same programmer content. If one station fails, the other station are not affected and continue working without loosing anything. This operator will usually have full rights to the entire system

    Multi User

    A multi-user operating system extends the basic multi-tasking concept with facilities that identify processes and resources, such as disk space, which belong to multiple users, and the system allows multiple users to interact with the system simultaneously. Time-sharing operating systems schedule tasks for efficient system utilization and may include accounting software for cost allocation of processor time, mass storage, printing and other resources for multiple users

    Differnce betweem Single-user, single task and Single-user, multi-tasking

    Single-user, single task Single-user, multi-tasking
    As the name implies, this operating system is designed to manage the computer so that one user can effectively do one thing at a time. The Palm OS for Palm handheld computers is a good example of a modern single-user,single-task operating system. This is the type of operating system most people use on their desktop and laptop computers today. Microsoft's Windows and Apple's MacOS platforms are both examples of operating systems that will let a single user have several programs in operation at the same time. For example, it's entirely possible for a Windows user to be writing a note in a word processor while downloading a file from the Internet while printing the text of an e-mail message.
