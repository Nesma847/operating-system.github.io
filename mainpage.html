Introduction

An operating system (OS) is machine software that manage computer hardware, computing resources, and provides that computer programs services.Time-sharing operating systems schedule activities for effective system use and may also provide accounting tools for processor time allocation, data storage, printing and other resources

The operating system serves as an intermediary between programs and computer hardware for hardware functions such as input and output, and memory allocation, while the application code is typically performed directly by the hardware and sometimes makes system calls to or is interrupted by an OS task.Operating systems are found on many devices that contain a computer â€“ from cellular phones and video game consoles to web servers and supercomputers.
